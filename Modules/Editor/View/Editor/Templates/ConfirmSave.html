Config File Editor Plugin will creates /wp-config-wcfe_backup.php from wp-config.php file just before removing its content

Please copy and past the following link to safe place, use it if you having issues access Wordpress after updating Config File. The following link is only available if COnfig Editor Plugin detects Wordpress stopped and only available for one time use

The link below will immediately restore back previous wp-config.php file.

This process will replace wp-config.php file content with content generated from the values you've entered in the form.

The process is totally replace the old code with new code generated by Config File Editor Plugin and all the code you had in wp-config.php will be replaced.

This is risky process and might stop Wordpress installation

This process never touch your Database or any file except wp-config.php file, if Wordpress stopped you just need to revert wp-config.php file back to old values

Please don't keep if you care about one of the following reasons:

  • wp-config.php contains custom code -- not originally bundled with normal Wordpress installation
  • You're not 100% positive of the values you changed
  • You manually added wp-config.php constants that not supported by Config File Editor plugin
